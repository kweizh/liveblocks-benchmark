### Implementation (`/home/user/myproject/app/page.tsx`)

The page component uses Liveblocks hooks for realtime collaborative state:

1. **Read todos** — `useStorage((root) => root.todos)` resolves the `LiveList<LiveObject<{id, text, done}>>` into a plain readonly array for rendering.

2. **Add todo** — `useMutation` pushes a new `LiveObject` with `crypto.randomUUID()` as `id`, the trimmed text, and `done: false`. The input clears after successful add. Empty text is rejected.

3. **Toggle done** — `useMutation` finds the todo by `id` and flips `done` on the underlying `LiveObject` via `item.set("done", !item.get("done"))`.

4. **Delete todo** — `useMutation` finds the todo by `id` and removes it from the `LiveList` via `list.delete(i)`.

5. **Clear completed** — A single `useMutation` callback iterates the `LiveList` backwards (to avoid index shifting) and deletes every item where `done === true`. This batches all deletions into one network operation so other clients see them atomically.
