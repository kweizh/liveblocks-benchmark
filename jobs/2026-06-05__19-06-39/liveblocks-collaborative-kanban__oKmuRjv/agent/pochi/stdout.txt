# Liveblocks Collaborative Kanban Board (Next.js) ## Background This Next.js project contains a scaffold for a real-time collaborative Kanban board powered by https://liveblocks.io/. The board has three fixed columns (`To Do`, `In Progress`, `Done`) and stores its data inside a shared Liveblocks room so that any change made by one user is instantly synchronized to every other user in the same room. The scaffold already renders the empty columns and wires up the `RoomProvider`, but it does not yet implement card creation, editing, deletion, or drag-and-drop movement between columns. Your job is to finish the Kanban application using **Liveblocks Storage** (`LiveList`, `LiveObject`, `LiveMap`) and the `useMutation` hook. The Liveblocks **cloud service is used directly** — there is no mocking.
